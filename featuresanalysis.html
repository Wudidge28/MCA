Using Time-Frequency based analysis over Waveform?

Time-Frequency analysis refers to the usage of Spectograms to study both Time and Frequency simultaneously. The use of Time-Frequency is different from waveform-based analysis in that it allows for the capture of multiple variations throughout a track. At a glance for example, we can see the track “66666666666666666666_66666666666666666666_66666666666666666666_66666666666666666666_66666666666666666666_66666666666666666666” by Monplaisir has multiple peaks and dips on the waveform above.
This allows for a good visual overview of the specificities of a track. However, using Time-Frequency analysis like we see in a spectrogram from the track above, we are able to better discern a general range of sound rather than simply highlighting short periods which are not significant. From this Spectrogram of Monplaisir’s track we can easily see the consistent level of sound throughout and therefore Time-Frequency visualisations like this one offer a more comprehensive analysis than waveform-based analysis.
